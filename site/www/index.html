üîß Quick Start

From the project root directory:

  • ./bin/init - Initialize project structure
  • ./bin/site status - Check project status
  • ./bin/site build - Build from source
  • ./bin/site serve - Start local web server
  • ./bin/site deploy - Create deployment package

Python Commands

  • python3 src/main.py status - Show project status
  • python3 src/main.py build - Python build process
  • python3 src/main.py deploy - Create deployment package

Make Commands

  • cd src && make build - Build project
  • cd src && make deploy - Deploy project
  • cd src && make clean - Clean build artifacts
